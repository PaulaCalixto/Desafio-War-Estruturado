# Desafio WAR Estruturado – Tema 1

Projeto simples em linguagem C simulando o jogo WAR de forma estruturada.

## Como compilar

Use `gcc`:

```bash
gcc main.c guerra.c -o war
./war

Regras básicas:

6 territórios.

2 jogadores.

Cada território começa com alguns exércitos.

O jogador ataca territórios vizinhos lançando dados.

Quem conquistar todos os 6 territórios vence.
